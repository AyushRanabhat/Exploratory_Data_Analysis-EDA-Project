# SQL Data Analytics Project

A comprehensive SQL-based project demonstrating end-to-end data exploration, analysis, and reporting. This repository includes a set of well-organized scripts that cover database exploration, dimension and date analysis, measures and metrics, segmentation, ranking, performance evaluation, cumulative analysis, and reporting.

This project uses publicly available retail sales data and is structured to help data analysts and BI professionals explore data, generate insights, and practice advanced SQL techniques.

---

## # Exploratory_Data_Analysis-EDA-Project

Exploratory analysis of publicly available retail sales data using SQL. Includes product and customer segmentation, sales trends, revenue analysis, rankings, and key business metrics for practice and learning purposes.

## ğŸ” Key Features

- **Database Exploration**: Explore tables, columns, key constraints, and relationships.  
- **Dimension Exploration**: Inspect categories, customers, and products; detect dimension changes.  
- **Date Analysis**: Examine order, shipping, and due dates; calculate gaps and customer ages.  
- **Measures and Metrics**: Compute total sales, quantity, average price, total orders, products, and customers.  
- **Magnitude Analysis**: Segment by cost, spending, and other business-relevant thresholds.  
- **Ranking Analysis**: Identify top/bottom products and customers based on revenue, quantity, or frequency.  
- **Change Over Time**: Month-over-month and year-over-year performance trends.  
- **Cumulative Analysis**: Running totals and averages for sales and price metrics.  
- **Performance Analysis**: Compare sales to averages and previous periods to evaluate consistency.  
- **Part-to-Whole Analysis**: Assess contributions of categories/products to overall sales.  
- **Segmentation**: Customer and product segmentation for targeted insights.  
- **Reports**: Ready-to-use SQL reports for products and customers.
